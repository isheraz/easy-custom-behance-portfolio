A Simple Plugin to display your portfolio on your website or blog

== Description ==

A  Simple Plugin to display your portfolio on your website or blog. The Plugin uses Behance API

Major features include:

* No API key required
* Select number of projects to display
* Linked to original posts on Behance
* Easily customizable for developers

== Installation ==

Upload the plugin to your blog, Activate it, then enter your behance username and setup display settings.

Display it anywhere using this shortcode [behanceportfolio]

And You're done! Good to go
